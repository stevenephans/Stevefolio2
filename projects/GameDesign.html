Unity Engine Game Design

C# + Unity Game Engine + Graphic Design

Unfortunately, the COVID-19 pandemic caused my prospective Summer internship to be cancelled so I needed to find a productive way to spend my, albeit unwanted, newfound free time. Confined to my apartment, my options were limited. I decided to undertake another coding project and signed up for a C#/.NET game development coures through Udemy.com. Progressing through 5 projects, the class served up a well-organized introduction to higher level language programming. I learned about data structures, class architectures, and object-oriented programming strategies.

I challenged myself to add custom features and functionality in each game design project, and even started building my own game from scratch after completing the course. While I would have preferred a pandemic-less Summer and the accompanying internship, I really enjoyed the course and learned a lot about higher level language programming.
