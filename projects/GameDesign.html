Unity Engine Game Design

C# + Unity Game Engine + Graphic Design

Unfortunately, the COVID-19 pandemic caused my prospective Summer internship to be cancelled so I needed to find a productive way to spend my, albeit unwanted, newfound free time. Confined to my apartment, my options were limited. I decided to undertake another coding project and signed up for a C#/.NET game development course through Udemy.com. Progressing through increasingly complex projects, the class served up a well-organized introduction to higher level language programming. I vastly improved my knowledge of data structures, class architectures, and object-oriented programming strategies.

I challenged myself to add custom features and functionality for each project, and even started building a game from scratch to improve and test my skills. While I would have preferred a pandemic-less Summer and accompanying internship, I enjoyed the course and greatly improved my higher level language programming abilities.
